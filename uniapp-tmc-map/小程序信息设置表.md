## 📝 功能介绍

### 核心功能页面
1. **首页地图**：显示俱乐部位置标记
2. **俱乐部详情**：查看具体信息
3. **筛选功能**：按时间筛选俱乐部

### 功能描述文案
```
【核心功能】
• 智能地图显示：在地图上直观显示所有头马俱乐部位置
• 详细信息查看：包括例会时间、地址、联系方式等完整信息  
• 智能筛选功能：可按例会时间（周一至周日）筛选合适的俱乐部
• 一键导航服务：支持调用地图应用进行路线导航
• 实时定位服务：自动定位用户位置，推荐附近俱乐部

【适用人群】
• 头马国际会员
• 演讲爱好者
• 寻找练习机会的职场人士
• 新加入头马的会员
```
